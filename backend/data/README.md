# Data Directory

Place your `sales_data.json` file here.

## File Format

The JSON file should contain an array of sales records with the following structure:

```json
[
  {
    "Customer ID": "CUST0001",
    "Customer Name": "John Doe",
    "Phone Number": "9876543210",
    "Gender": "Male",
    "Age": 35,
    "Customer Region": "North",
    "Customer Type": "Regular",
    "Product ID": "PROD0001",
    "Product Name": "Laptop",
    "Brand": "Dell",
    "Product Category": "Electronics",
    "Tags": "Premium, Sale",
    "Quantity": 2,
    "Price per Unit": 50000,
    "Discount Percentage": 10,
    "Total Amount": 100000,
    "Final Amount": 90000,
    "Date": "2024-01-15",
    "Payment Method": "Credit Card",
    "Order Status": "Completed",
    "Delivery Type": "Home Delivery",
    "Store ID": "STORE1",
    "Store Location": "Mumbai",
    "Salesperson ID": "EMP001",
    "Employee Name": "Jane Smith"
  }
]
```

## Note

If the file is not present, the system will automatically generate sample data for testing purposes.
